# Sovereign Cloud Engine (Elite-Tier)

An enterprise-grade, high-availability cloud operations dashboard. Engineered with a heavily decoupled architecture using React, Node.js, Docker, and Terraform for automated AWS infrastructure provisioning.

## 🚀 Features
*   **100% Modular Architecture:** Fully componentized React frontend (10 active side-bar modules).
*   **Geographic Middleware:** Custom Node.js/Express interceptors simulating cross-border data sovereignty (GDPR, DPDP Act) via header manipulation.
*   **Executive AI Integrity Suite:** Automated infrastructure scanning protocol confirming structural isolation.
*   **Cyber-Dark Aesthetic:** Premium `backdrop-filter` glassmorphism and an interactive cyan cursor-glow.

## 🛠️ Tech Stack
*   **Frontend:** React (Vite), CSS3 Variables, Component-based state routing.
*   **Backend:** Node.js, Express, AWS SDK.
*   **Infrastructure:** Terraform (HCL), LocalStack (Mock DynamoDB), Docker Compose.
